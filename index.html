AIA Forex System AI trading assistant · MT5
Multi-layered Forex AI with risk brain
A “Jarvis-style” assistant built on MetaTrader 5: real-time data ingestion, feature builder, probabilistic forecasts, and a strict risk policy (R-based sizing, loss limits, observe-only modes) wrapped in a FastAPI backend and React UI.
Python · FastAPI MetaTrader5 API ML & risk models
NeuroSync / Senceteck Immersive Reality · BCI
Recording & replaying subjective experience
Concept and architecture for a system that captures multi-modal streams (EEG/BCI, biometrics, video, audio) into a structured .EXP format. Focus on synchronisation, emotional interpretation and safe replay of lived experiences.
BCI · EEG Python services Experience file format
EmotionSync iOS app · IR
Global Emotion Map & emotional library
iOS prototype that lets users record emotional episodes, analyse them with AI (video, audio, face/emotion tracking) and place them on a global map. Includes emotion library with previews and an interpreter that packs everything into .EXP.
SwiftUI Gemini API MediaPipe · JSON pipelines
